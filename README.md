# Onboarding Tracker
Проект: Onboarding Tracker

Описание проекта:
Веб-приложение для хранения и отслеживания прогресса сотрудников при прохождении обучающих материалов предприятия.
Администраторы могут добавлять модули обучения, а сотрудники отмечают пройденные материалы.

Функциональные требования:
1.Регистрация и авторизация:
     1.Форма регистрации: имя, email, пароль, роль (сотрудник/администратор).
     2.Форма входа: проверка email и пароля.
     3.Сессия или токен для защиты данных.
2.Личный кабинет:
    1.Сотрудник видит список всех доступных модулей обучения.
    2.Для каждого модуля:
          -название,
          -краткое описание,
          -категория (например: безопасность, IT, процессы).
    3.Возможности:
          -отметить модуль как пройденный
          -видеть прогресс (например, количество пройденных модулей / общее количество).
3. Роли и права доступа:
     1.Администратор: может добавлять, редактировать, удалять модули обучения, смотреть прогресс всех сотрудников.
     2.Сотрудник: может отмечать пройденные модули и просматривать свой прогресс.
     3.Неавторизованные пользователи перенаправляются на страницу входа.
4.Категории обучения:
     1.Возможность создавать категории модулей (например, «Безопасность», «IT», «Процессы»).
     2.Фильтрация модулей по категориям.
5.Минимальный дизайн:
     1.Интерфейс аккуратный и понятный.
     2.Наличие:
          -шапки (Header) с логотипом проекта
          -кнопки выхода (Logout)
          -основной страницы с модулями в виде карточек

Технологии:
-Фреймворк: Yii2
-PHP 
-База данных: MySQL (MariaDB 10.4.32)
-HTML, CSS, Js, Bootstrap 5
-Веб-сервер: Apache 2.4.58

Требования к системе:
- PHP 8.2+ 
- MariaDB 10.4+ или MySQL 8.0+
- Apache 2.4+
